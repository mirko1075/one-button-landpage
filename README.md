# InOneButton Landing Page

A React landing page for InOneButton - the simplest AI meeting companion.

## Features

- WhatsApp-style recording interaction with Apple-level polish
- Premium microphone button with ripple effects and heartbeat animations
- Live equalizer animation during recording
- Supabase integration for waitlist management
- Fully responsive design
- Modern React + Vite setup

## Getting Started

### Install dependencies
```bash
npm install
```

### Run development server
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

### Preview production build
```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Supabase
- Pure CSS animations
